Word quiz
Getest in IDLE en Python shell

**Doel van het script is om woordenschat in te oefenen. Kan ook gebruikt worden om andere soorten QA oefeningen te maken.
Animals.csv = test file
Word quiz = script

Het script bestaat uit 3 delen:

- Openen of aanmaken van files
- Aanpassen van files
- Quiz

*Openen of aanmaken van files (Files moeten in dezelfde map staan als het script:
- Optie om een nieuwe file te maken
	Bestaat de file? optie om te overwriten of een andere naam te kiezen.
- Optie om een bestaande file te openen	
	Bestaat de file niet? optie om deze aan te maken of opnieuw proberen met de juiste naam.

*Aanpassen van files
	Als de inhoud van de files leeg is, is deze stap verplicht.
- Items verwijderen
	Optie om woordparen te verwijderen.
- Items toevoegen
	Optie om woordparen toe te voegen.
- Opslaan / Exit zonder opslaan / terugkeren naar het hoofdmenu
	Optie om de woordenlijst op te slaan in het csv bestand
	Optie om door te doen zonder op te slaan. (Niet mogelijk als het bronbestand leeg is).

*Quiz
- Opties om de woorden in te oefenen (shuffle, alfabetisch, standaardvolgorde)
- Quiz

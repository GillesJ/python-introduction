De file die geopend moet worden is gamecarousel.py. Dit is de 'home' van vijf aparte spelletjes.

In het begin van de carousel krijg je 10 credits, waarmee je een spel kunt betalen. 
Hoe groter en/of moeilijker het spel, hoe groter de inkoopprijs, maar ook hoe groter de (mogelijke) reward.

Er is geen limiet voor de mogelijk te behalen credits, maar als je geen credits meer overhebt, dan sluit de carousel.
